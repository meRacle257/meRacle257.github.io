Hexo+Github搭建博客

技术 教程
 2018/01/31   Share

记录自己搭建博客的过程，Mac环境。

Github

首先创建自己的Github Page，也就是创建名为 username.github.io的repository。然后什么都不用做了。

Hexo

  1. 安装nodejs，上官网，选择x.x.x.LTS版本下载，这个是稳定版，右边的x.x.x.CURRENT是最新版，有几率会出问题。

  2. 安装和配置Hexo

    npm install -g hexo-cli --no-optional
//cd至想要保存博客地址的文件夹，执行以下命令
//初始化
hexo init 
//编译
npm install
//启动
hexo server

    经过了以上几步就创建成功了，如果没什么问题的话，localhost:4000就可以访问本地的博客了。

  3. 创建页面

    //创建一个名字为pageName的页面
hexo new "pageName"

然后打开source/_post目录就可以看到md文件，尽情的编辑吧，保存之后你就会发现文章出现在了刚才打开的本地博客中。以后可以不用命令创建，直接把md文件放在这里就可以了。

上传至Github

  1. 安装插件

    npm install hexo-deployer-git --save

  2. 修改_config.yml

    deploy: 
    type: git 
    repo: https://github.com/用户名/用户名.github.io.git 
    branch: master

  3. 上传

    hexo clean
hexo g
hexo d

    执行完以上所有步骤之后如果没有问题就可以在自己的【用户名.github.io】网址上看到刚才在本地看到的博客啦！

其他

到这里已经正常部署啦，但是还有一些其他优化的东西。

自定义域名

  1. 购买域名并配置：以阿里云为例，购买域名后，配置两个A记录，记录地址@和WWW，IP地址就是ping自己的GithubPage的地址。

  2. GIthub自己的Page项目根目录下添加CNAME文件，不需要任何后缀，其中写上自己购买的域名即可。

主题

戳这里进入选择主题，克隆下来，按照主题给出的教程操作即可。

总结

大爱hexo，真的又漂亮又简单，再也不用自己写了，专心搞算法～
